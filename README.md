# GUI for 0x0.st

Uploads a chosen file to the 0x0.st and returns the link for it.

Start the program:

## 2.

## 3.
Click on Upload:
![Screenshot 1](/misc/img/screen3.png)

## 4.
Now you can see your generated 0x0.st link:
